# Install Node.js

This section explains how to install Node.js on macOS.

---

## Install Using Homebrew

Run the following command:

```bash
brew install node
```

---

## Verify Installation

Check the Node.js version:

```bash
node -v
```

Check npm version:

```bash
npm -v
```

Example output:

```
v20.x.x
10.x.x
```
