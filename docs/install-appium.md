# Appium Installation Guide (macOS)

## Overview

This guide explains how to install and configure **Appium** on macOS for mobile test automation.

Appium is an open-source automation framework that allows testers and developers to automate native, hybrid, and mobile web applications on platforms such as **iOS** and **Android**.

This document covers:

- Installing required dependencies
- Installing Appium using Node.js
- Verifying the installation
- Installing platform drivers
- Running the Appium server

---

## Target Audience

This guide is intended for:

- QA Engineers
- Test Automation Engineers
- Mobile Testers
- Developers performing mobile application testing

---

## Supported Environment

| Component | Version |
|-----------|--------|
| Operating System | macOS |
| Node.js | v16 or later |
| Appium | Latest |
